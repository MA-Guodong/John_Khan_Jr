# 小约翰可汗的签名与头像

## 1.项目介绍

这是由`nodeJS`完成的项目，通过读取数据库中的数据，将其展示到网页上，包括头像、签名以及视频列表。后面有空可能会更新其他的内容，比如数据分析（图表形式）等

2023.11.13：数据可视化完成

数据库中的数据来自于`Python`脚本爬取自B站，将内容存入数据库中，脚本内容见[Python脚本](https://github.com/MA-Guodong/John_Khan_Jr_Python_Projects_MySQL.git)

## 2.Install 安装

`npm install`

## 3. Start 启动

`node index.js`
